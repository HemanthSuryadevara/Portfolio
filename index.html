// 01 — About

The person
behind the logic

I'm a Design Verification Engineer with a deep passion for catching bugs before they become recalls. I completed my M.S. in Electrical Engineering at California State University, Northridge, focusing on ASIC/VLSI verification methodologies.

Before grad school, I spent two years at Tata Elxsi as a Junior Embedded Systems Engineer, building production-grade firmware for automotive IoT systems — which gave me a rare perspective: I understand both the hardware I verify and the software that runs on it.

My thesis developed an improved nonlinear DC I-V model for GaN MESFETs, applying numerical methods and parameter sensitivity analysis in MATLAB — a project that sharpened my ability to think rigorously about real device behavior.

// 02 — Projects

Selected work

RTL verification, protocol implementation, and hardware modeling — a collection of projects spanning ASIC design verification, embedded systems, and transistor-level design.

// 01
UART/RS-232 Serial Communication Verification
Architected and verified a complete RTL-level RS-232 UART subsystem with full-duplex communication. FSM-based transmitter/receiver with baud-rate control (16x oversampling), 9.6k–115.2k baud support, and >95% code coverage across 50+ test scenarios.
SystemVerilog FSM Design Self-Checking TB Full Coverage
// 02
SPI Master-Slave Protocol Full-Duplex Verification
Comprehensive verification plan for an SPI subsystem across all 4 modes (CPOL/CPHA). Verified TX/RX shift register logic, identified timing path violations, drove RTL fixes achieving <5ns propagation delay. 100+ test scenarios with full functional coverage closure.
Verilog Synopsys VCS Protocol Verification Timing Analysis
// 03
GaN MESFET Nonlinear DC I-V Model (Thesis)
Developed an enhanced nonlinear I-V model for nanometer-scale GaN MESFETs with 15–20% improved accuracy over baseline. Jacobian-based parameter sensitivity analysis, RMSE <3% curve fitting, and validation across saturation, linear, and breakdown regions.
MATLAB Device Modeling RF Electronics GaN Technology
// 04
LFSR Design for Pseudo-Random Test Vector Generation
Designed and verified a 32-bit Galois-style LFSR with 2³²−1 period for BIST applications. Validated primitive polynomial feedback taps for maximal-length m-sequences. 100% functional correctness with exhaustive state coverage verification.
Verilog BIST DFT RTL Design
// 05
CMOS Multiplexer Transistor-Level Analysis
Designed a 2:1 CMOS mux at transistor level — both transmission-gate and NAND/NOR architectures. W/L ratio optimization study, PVT corner validation (10% delay variation), and power-delay product minimization for datapath circuit design.
CMOSedu LTspice SPICE Simulation CMOS Design
// 06
Automotive Adaptive Headlight Control System
Designed an adaptive headlight system for dynamic beam intensity control. Integrated ultrasonic and LDR sensors on Arduino for vehicle detection (up to 100m) and real-time ambient measurement. Distance-based intensity modulation with optimized sensor calibration.
Arduino C Embedded Firmware Automotive
